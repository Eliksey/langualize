# Langualize — Project Planning Notes

## Initial Brainstorming (October 2025)
- Problem: existing language apps lack real AI conversation practice
- Idea: combine multiple AI models (GPT-4, Claude, GLM-4) for adaptive dialogue
- Target audience: students 14-18 learning foreign languages
- Key differentiator: real-time grammar correction + conversational practice

## Goals Defined
- Learning Goal: Develop analytical and research skills necessary to investigate how AI-powered conversational practice can enhance foreign language acquisition
- Product Goal: Create a functional cross-platform app integrating 3 AI models

## Initial Feature List
1. AI conversation practice (multi-language)
2. Grammar correction with explanations
3. Vocabulary builder with spaced repetition
4. Progress tracking and lesson reports
5. Gamification (XP, streaks, achievements)
