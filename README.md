# Langualize — AI-Powered Language Learning Application

Individual school project by Elisei Alferov, Grade 9
International Gymnasium of the Skolkovo Innovation Center

## Overview
Langualize is a cross-platform mobile application that uses AI-powered conversational practice to enhance foreign language acquisition. The app integrates three AI models (GPT-4, Claude, GLM-4) for adaptive dialogue, real-time grammar correction, and vocabulary building.

## Tech Stack
- **Frontend:** Flutter v3 (Dart)
- **Backend:** NestJS v10 + Prisma v5
- **Database:** PostgreSQL 16
- **AI Models:** GPT-4, Claude (Anthropic), GLM-4

## Key Features
- AI conversation practice in multiple languages
- Real-time grammar correction with explanations
- Lesson performance reports with error analysis
- Gamification: XP, streaks, achievements
- Progress tracking dashboard

## Beta Testing Results (n=8)
- 87% would recommend the app
- 4.3/5 average report rating
- 85-90% grammar error detection accuracy
